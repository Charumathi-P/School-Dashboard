# 🎓 School Dashboard
A responsive and modern School Management Dashboard built using HTML, CSS, and JavaScript. The dashboard helps administrators manage students, teachers, attendance, fees, examinations, and other school-related activities through an intuitive interface.

---

## ✨ Features

- 🔐 Login Page
- 📊 Dashboard Overview
- 👨‍🎓 Student Management
- 👩‍🏫 Teacher Management
- 🏫 Class Management
- 📅 Attendance Tracking
- 📝 Examination & Results
- 💰 Fee Management
- 📚 Subject Management
- 📢 Announcements
- 🔍 Search Functionality
- 🌙 Dark Mode
- 📱 Fully Responsive Design

---

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

---

## 🚀 Installation
1. Clone the repository

```bash
git clone https://github.com/yourusername/School-Dashboard.git
```
2. Open the project folder.
3. Run `index.html` in your browser.
